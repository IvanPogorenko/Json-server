Json-server для имитации работы с REAST API.

для запуска использовать команды:

1. npm install
2. npm run server

Методы:

1. post('/login') - принимает body, в котором описаны login и password пользователя, проводит авторизацию и возвращает
   объект пользователя из БД
2. get('/markers') - объединяет маркеры и цвета по user_id, возвращая объект маркера из бд с полем color
3. post('/create-marker') - принимает объект маркера без id, создает id на основе последнего в бд и сохраняет запись
4. patch('/marker/:id') - принимает изменяемые поля в body и id из параметра запроса, после производит поиск и изменение
   соответствующей записи
5. delete('/marker/:id') - удаляет маркер по id
